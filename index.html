How to read MediShield SOC

CVE A public vulnerability identifier, like a tracking number for a software security issue.
CISA KEV A public catalog of vulnerabilities known to be exploited in the wild.
NVD The U.S. National Vulnerability Database, used for CVE details and CVSS severity.
SOC / SIEM A security operations workflow for collecting alerts, spotting risk, and triaging incidents.
Healthcare signal A simple relevance count from healthcare terms such as medical, hospital, patient, infusion, or radiology.
